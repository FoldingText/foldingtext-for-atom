A record of a single change in a target Item.

A new mutation is created to record each attribute set, body text change, and child itemâ€™s update. Use Outline::onDidChange to receive this mutation record so you can track what has changed as an outline is edited.

Constants

.ATTRIBUTE_CHANGED

ATTRIBUTE_CHANGED Mutation type constant.

.BODT_TEXT_CHANGED

BODT_TEXT_CHANGED Mutation type constant.

.CHILDREN_CHANGED

CHILDREN_CHANGED Mutation type constant.

Attributes

::target

Read-only Item target of the change delta.

::type

Read-only type of change. Mutation.ATTRIBUTE_CHANGED, Mutation.BODT_TEXT_CHANGED, or Mutation.CHILDREN_CHANGED.

::attributeName

Read-only name of changed attribute in the target Item, or null.

::attributeOldValue

Read-only previous value of changed attribute in the target Item, or null.

::insertedTextLocation

Read-only value of the body text location where the insert started in the target Item, or null.

::insertedTextLength

Read-only value of length of the inserted body text in the target Item, or null.

::replacedText

Read-only value of replaced body text in the target Item, or null.

::addedItems

Read-only Array of child Items added to the target.

::removedItems

Read-only Array of child Items removed from the target.

::previousSibling

Read-only previous sibling Item of the added or removed Items, or null.

::nextSibling

Read-only next sibling Item of the added or removed Items, or null.
