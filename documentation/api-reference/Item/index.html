--- layout: class title: Item breadcrumb: Item ---

A paragraph of text in an Outline.

Items cannot be instantiated directly, instead use Outline::createItem.

Items may contain other child items to form a hierarchical outline structure. When you move an item all of its children are moved with it.

Items have a single paragraph of body text. You can access it as plain text, a HTML string, or an AttributedString. You can add formatting to make parts of the text bold, italic, etc.

You can assign item level attributes to items. For example you might store a due date in the data-due-date attribute. Or store an item type in the data-type attribute.

Examples

Create Items: 
item = outline.createItem('Hello World!')
outline.root.appendChild(item)
Add body text formatting: 
item = outline.createItem('Hello World!')
item.addElementInBodyTextRange('B', {}, 6, 5)
item.addElementInBodyTextRange('I', {}, 0, 11)
Read body text formatting: 
effectiveRange = end: 0
textLength = item.bodyText.length
while effectiveRange.end < textLength
  console.log item.getElementsAtBodyTextIndex effectiveRange.end, effectiveRange

Attributes

::id

Read-only unique and persistent String item ID.

::attributeNames

Read-only Array of this item’s attribute names.

::hasAttribute(name)

Test to see if this item has an attribute with the given name.

Argument Description
name The String attribute name.
Return Values
Returns a Boolean
::getAttribute(name, array?, clazz?)

Get the value of the specified attribute. If the attribute does not exist will return undefined.

Argument Description
name The String attribute name.
array? Boolean true if should split comma separated value to create an array.
clazz? {Class} (Number or Date) to convert string values into.
Return Values
Returns attribute value.
::setAttribute(name, value)

Adds a new attribute or changes the value of an existing attribute. id is reserved and an exception is thrown if you try to set it. Non string values (such as Dates) will be converted to appropriate string format so that they can be read back using ::getAttribute(). Setting an attribute to null or undefined will remove the attribute.

Argument Description
name The String attribute name.
value The new attribute value.
::removeAttribute(name)

Removes an attribute from the specified item. Attempting to remove an attribute that is not on the item doesn’t raise an exception.

Argument Description
name The String attribute name.

Body Text

::hasBodyText

Read-only true if this item has body text

::bodyText

Body text as plain text String.

::bodyHTML

Body text as HTML String.

::attributedBodyText

Body text as AttributedString.

::getAttributedBodyTextSubstring(location, length)
Argument Description
location Substring’s strart location.
length Length of substring to extract.
Return Values
Returns an AttributedString substring of this item’s body text.
::getElementAtBodyTextIndex(tagName, index, effectiveRange?, longestEffectiveRange?)

Looks to see if there’s an element with the given tagName at the given index. If there is then that element’s attributes are returned and by reference the range over which the element applies.

Argument Description
tagName Tag name of the element.
index The character index.
effectiveRange? Object whose location and length properties will be set to effective range of element.
longestEffectiveRange? Object whose location and length properties will be set to longest effective range of element.
Return Values
Returns elements attribute values as an Object or {}
::getElementsAtBodyTextIndex(index, effectiveRange?, longestEffectiveRange?)
Argument Description
index The character index.
effectiveRange? Object whose location and length properties will be set to effective range of element.
longestEffectiveRange? Object whose location and length properties will be set to longest effective range of element.
Return Values
Returns an Object with keys for each element at the given character index, and by reference the range over which the elements apply.
::addElementInBodyTextRange(tagName, attributes, location, length)

Adds an element with the given tagName and attributes to the characters in the specified range.

Argument Description
tagName Tag name of the element.
attributes Element attributes. Use null as a placeholder if element doesn’t need attributes.
location Start location character index.
length Range length.
::removeElementInBodyTextRange(tagName, location, length)

Removes the element with the tagName from the characters in the specified range.

Argument Description
tagName Tag name of the element.
location Start location character index.
length Range length.
::replaceBodyTextInRange(insertedText, location, length)

Replace body text in the given range.

Argument Description
insertedText String or AttributedString
location Start location character index.
length Range length.
::appendBodyText(text, elements?)

Append body text.

Argument Description
text String or AttributedString
elements? Object whose keys are formatting element tagNames and values are attributes for those elements. If specified the appended text will include these elements.

Given an array of items determines and returns the common ancestors of those items.

Argument Description

Return Values

::contains(item)

Determines if this item contains the given item.

Argument Description
item The Item to check for containment.
Return Values

::comparePosition(item)

Argument Description
item The Item to compare against.
Return Values

::cloneItem()

Deep clones this item.

Return Values
Returns a duplicate Item.

Mutating Outline Structure

::insertChildBefore(child, referenceSibling?)

Insert the new child item before the referenced sibling in this item’s list of children. If referenceSibling isn’t defined the item is inserted at the end.

Argument Description
child The inserted child Item .
referenceSibling? The referenced sibling Item .
::insertChildrenBefore(children, referenceSibling?)

Insert the new children before the referenced sibling in this item’s list of children. If referenceSibling isn’t defined the new children are inserted at the end.

Argument Description
children Array of Items to insert.
referenceSibling? The referenced sibling Item.
::appendChildren(children)

Append the new children to this item’s list of children.

Argument Description
children The children Array to append.
::appendChild(child)

Append the new child to this item’s list of children.

Argument Description
child The child Item to append.
::removeChildren(children)

Remove the children from this item’s list of children.

Argument Description
children The Array of children {Items}s to remove.
::removeChild(child)

Remove the given child from this item’s list of children.

Argument Description
child The child Item to remove.
::removeFromParent()

Remove this item from it’s parent item if it has a parent.
