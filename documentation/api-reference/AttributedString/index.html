A container holding both characters and formatting attributes.

AttributedStrings have a limited public API. Right now they are mostly useful as containers for moving text and attributes from one Item‘s body text to another item’s body text. If you need to edit or list an item’s text attributes use the item attribute methods instead:

  • Item::getAttributedBodyTextSubstring
  • Item::getElementAtBodyTextIndex
  • Item::addElementInBodyTextRange
  • Item::removeElementInBodyTextRange
  • Item::replaceBodyTextInRange
